# Doctor Web Project

This repository contains a full-stack **Doctor Appointment Web Application**.  
It consists of:  

- **Backend:** `doctor-app-new` (Laravel API)  
- **Frontend:** `doctor-frontend` (React / Vue / JS client)  
- **Database:** PostgreSQL
